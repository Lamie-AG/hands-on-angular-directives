Angular Directives Example

In this example we will have a look a directives. Angular is shipped with plenty of pre-built directives like NgModel, NgClass, NgIf, NgFor.
We differentiate between structural directives like NgIf, NgFor, NgSwitch and between attribute directives like NgModel, NgClass, NgStyle.
Note that all of the listet directives above are pre-builtin but we can still create custom ones depending on use cases.
